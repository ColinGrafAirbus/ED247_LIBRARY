LIBED247_EXPORT ed247_status_t libed247_register_set_simulation_time_ns_handler ( libed247_set_simulation_time_ns_t  handler,
void *  user_data 
)

Register a callback used to timestamp sample at recpection (recv_timestamp) The registered function is called each time the simulation time is needed, in each stream receiving data. It is strongly encouraged to perform a manual increase of simulation time and not clock queries each time the function is called as this might lead to a high execution time.

Parameters
[in]handlerHandler to the function
[in]user_dataPointer to custom data
Return values
ED247_STATUS_SUCCESS
ED247_STATUS_FAILURE
