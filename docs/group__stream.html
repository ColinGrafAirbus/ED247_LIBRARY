LIBED247_EXPORT ed247_status_t ed247_find_channel_streams ( ed247_channel_t  channel,
const char *  regex_name,
ed247_stream_list_t *  streams 
)

Find all streams of the channel whose name is matching the regular expression For example, to get a list of all the streams, use the /a * value. Do not use during runtime. The implementation may contain memory allocation functions.

Parameters
[in]channelThe channel identifier
[in]regex_nameThe regular expression for name matching. If null, assume '.*'.
[out]streamsThe list of the streams. If no value, set to null.
Return values
ED247_STATUS_SUCCESS
ED247_STATUS_FAILUREThe stream list is empty
Note
The regular expressions are not implemented in gcc4.8.x (default linux), do not use complex requests.
